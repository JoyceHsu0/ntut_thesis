基於反向傳播映射醫學影像分割張量立體成像

Backpropagation-based Mapping Medical Image Segmentation Tensor Volume Rendering

  • 前往網頁 GitHub 前往系統 GitHub

摘要

本研究旨在提升三維醫學影像分割模型的可解釋性，提出一套結合梯度加權類別激活映射與三維立體成像技術的可視化分析框架。本系統提供逐層特徵可視化功能，使用者可針對三維分割模型 UNetCNX 中任意層級與通道生成對應的梯度加權類別激活映射熱力圖，以深入分析模型的判斷依據。 為強化系統彈性與運算效能，亦對 UNetCNX 模型架構與推論流程進行最佳化，並將熱力圖生成與三維繪製流程模組化分離，支援即時調整視角與呈現風格。整體設計不僅提升模型可解釋性與互動性，亦提供穩定高效的可視化系統，促進深度學習於醫學影像分析中的可信與實用應用。
